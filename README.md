## Stack

Single-page static site, no build step:

- **HTML5** + semantic landmarks
- **Tailwind CSS** (via CDN) for styling
- **Vanilla JS** for the mobile menu, smooth scroll, and the `IntersectionObserver` reveal animations
- **GitHub Pages** for hosting (deployed straight from `main`)

## Local Development

```bash
git clone https://github.com/Chetas1/chetas1.github.io.git
cd chetas1.github.io

# Any static server works; pick one:
python3 -m http.server 8000
# or:
npx serve .
```

Then open `http://localhost:8000`.

## Deploying

Pushes to `main` are published automatically by GitHub Pages. The site root is the repository root (`index.html`); there is no build step.
